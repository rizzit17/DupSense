DupSense is a premium desktop utility that finds exact duplicates, visually similar images, and highly identical text files. All processing is 100% local. Your files never leave your machine.

Version 1.0.0 • Requires Windows 10/11

Pro-Level Features

Engineered for accuracy and safety.

01 // AI Image Diffing

Side-by-Side Comparison

Uses perceptual hashing (pHash) to find images that are visually identical, even if compressed, resized, or watermarked. Blow them up side-by-side to make the final call.

02 // Safe Zones

Absolute Directory Protection

Declare critical paths like `C:\Windows`. DupSense will scan them to find duplicates elsewhere, but will actively block you from ever deleting the protected originals.

03 // Text Similarity

Fuzzy Code Matching

Powered by thefuzz. Scans code files, markdown, and essays to detect versions that are 90% identical in content, catching the "final_final_v2.txt" clutter.

04 // Data Visualization

Interactive Treemaps

Stop guessing where your storage went. View massive Plotly-powered Treemaps that visualize your directory hierarchy and pinpoint exactly which folders are hoarding duplicates.
